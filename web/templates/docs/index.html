1. Dashboard Utama

Apa fungsi halaman Dashboard?

Dashboard adalah halaman pertama saat Admin login. Berfungsi untuk merangkum status seluruh bisnis Anda secara sekilas:

  • Total pelanggan aktif, terisolir (menunggak), dan total router yang online.
  • Terdapat 3 tombol peringatan cepat: Pelanggan Menunggak (Tunggakan), Tagihan Lunas Bulan Ini, dan peringatan Router Offline.
  • Akses cepat ke portal klien dan teknisi.

2. Internet Rumahan (PPPoE)

Bagaimana Cara Mendaftarkan Pelanggan Bulanan Baru?
  • Pertama, pastikan Anda sudah menambahkan Paket Internet (Profil PPPoE) di menu Master Data -> Daftar Profil.
  • Buka menu Internet Rumahan -> Data Pelanggan.
  • Klik tombol biru + Tambah Pelanggan Baru di sudut kanan atas.
  • Isi informasi PPPoE (Username & Password harus sama dengan konfigurasi di Modem Pelanggan), pilih paket langganan, dan tentukan TANGGAL JATUH TEMPO.
  • Klik Simpan. Sistem MikroFun akan otomatis membuatkan Secret PPPoE pada mikrotik Anda (jika Auto-Sync aktif).
Bagaimana Cara Mengisolir (Blokir) Pelanggan Menunggak?

MikroFun Radius mendukung dua cara:

  • Otomatis: Aplikasi memiliki Cron Job (Background Task) yang secara konstan mengecek tanggal jatuh tempo. Jika pelanggan belum membayar hingga H+1 tenggat waktu, sistem otomatis mengganti Profil Mikrotik mereka ke profil isolir (kecepatan 0 atau dibelokkan ke web peringatan).
  • Manual: Di Data Pelanggan, klik tombol Aksi -> Paksa Isolir. Kapanpun Anda mau, koneksi mereka akan langsung diputus. Untuk membuka blokir, perbarui status pembayarannya atau klik tombol Aksi -> Buka Isolir.

3. Internet Voucher (Hotspot)

Bagaimana Cara Membuat Kode Voucher (Generate Batch)?
  • Pastikan ada paket voucher di Master Data -> Daftar Profil (Tipe: Voucher).
  • Pergi ke menu Internet Voucher -> Data Voucher.
  • Klik Generate Voucher. Masukkan jumlah voucher yang ingin dibuat (misal 50 lembar), beserta format karakternya.
  • Setelah digenerate, mikrotik akan menerima 50 data User Hotspot baru, dan Anda akan dialihkan ke halaman Print/Cetak untuk menggunting kode voucher fisik.
Bagaimana Klien Membeli Voucher Secara Online?

Aplikasi ini memiliki E-Commerce Bawaan di Portal Klien.

  • Klien mengakses Web Landing Page dan masuk ke Portal Klien (membuat akun sendiri).
  • Klien membuka menu Beli Voucher, dan memilih Paket Hotspot yang tersedia.
  • Klien membayar (Via QRIS/Transfer Midtrans atau Transfer Manual).
  • Jika via Midtrans, voucher akan otomatis dikirim (Di Generate "On The Fly") ke layar Klien seketika setelah pembayaran sukses. Jika manual, Anda harus memverifikasi bukti transfernya di menu Riwayat Penjualan -> Setujui agar Klien mendapatkan kodenya.

4. Tagihan & Pembayaran (Billing)

Bagaimana Sistem Pembayaran Bekerja?

Ada dua metode utama bagi pelanggan PPPoE untuk melunasi tagihan bulanan:

  1. Otomatis lewat Payment Gateway (Midtrans/Tripay): Direkomendasikan! Jika Klien membayar lewat portal menggunakan VA/QRIS, Gateway akan mengirim laporan (Webhook) ke MikroFun secara instan. MikroFun akan mengubah status klien dari "Nunggak" menjadi "Aktif", memperpanjang masa aktif 1 bulan penuh, dan menyalakan kembali internetnya meski Admin sedang tidur.
  2. Upload Bukti Transfer Manual: Klien mentransfer ke BCA pribadi Admin, memfoto resi, lalu mengunggahnya. Admin dapat melihat di Billing -> Konfirmasi Manual. Admin klik ikon "Mata" lalu pilih "Terima". Koneksi klien akan otomatis pulih.

5. Manajemen Jaringan (Router & ODP)

Menghubungkan & Konfigurasi Data Router / NAS

Menu Data Router / NAS adalah jantung sistem MikroFun.

  • Klik Tambah Router untuk mendaftarkan Mikrotik baru yang terletak di pusat atau cabang.
  • Isi IP Address (IP Publik atau Tunnel VPN Anda), Username, dan Password Mikrotik. Port API default adalah 8728 (atau 8729 untuk API-SSL).
  • Sistem ini menggunakan metode API Mikrotik untuk mendorong data secara sinkronis. Pastikan Layanan API pada menu `IP > Services` di Mikrotik Anda sudah diaktifkan.
  • Centang opsi "Status Aktif" jika Anda sudah yakin. Sistem hanya akan menembakkan konfigurasi (seperti Secret PPPoE baru & otomatisasi Isolir) ke Router yang statusnya Aktif.
  • Anda bebas memiliki lebih dari satu Router pada sistem MikroFun ini. Saat membuat Pelanggan Baru, pastikan Anda menempatkannya di Router yang benar.
Cara Membaca & Menambah Data ODP (Optical Distribution Point)

Fitur Data ODP / Peta memungkinkan ISP memetakan tiang-tiang distribusi Fiber Optic secara visual di atas peta (Google Maps API / OpenStreetMap).

  • Menambah ODP: Buka Peta ODP, tentukan lokasi titik koordinatnya di Peta. Isikan Kode ODP (misal `ODP-01-Maju`), Rasio Splitter (misal 1:8 atau 1:16), dan Warna Kabel (Tube).
  • Memasukkan Pelanggan: Saat mendaftarkan Pelanggan Baru di menu PPPoE, Anda bisa memilih ODP mana ia disambungkan, beserta Port Nomor ke-berapa.
  • Kegunaan: Saat terjadi `Red-Loss` atau kabel putus, Anda / Teknisi bisa langsung melihat rute kabel dan koordinat pasti kotak ODP mana di lapangan yang perlu dicek. Ini akan menghemat jam investigasi teknisi di jalanan raya!
Catatan Premium: Fitur Peta ODP dan GPS Visual mungkin terkunci bagi pemegang versi Free.

6. Web Pelanggan (Landing Page)

Manajemen Landing Page Public

Aplikasi menyediakan Website Promosi yang terbuka untuk publik pada alamat root `/` jika bukan Admin.

  • Edit teks promosi, alamat WiFi, dan daftar paket internet di menu Settings -> Konfigurasi Landing Page.
  • Sistem memiliki 3 Template Tema Default, dan opsi mengunggah HTML/CSS sendiri melalui Custom Template.

7. Helpdesk & Teknisi Lapangan

Alur Tiket Bantuan & Sistem Dispatch (Penugasan Teknisi)

Bagaimana sebuah komplain diproses secara profesional menggunakan Sistem Dispatch Lapangan?

  1. Laporan Diterima: Klien mengajukan keluhan (misal: "LOS Merah") melalui fitur Bantuan (Tiket) di Portal Klien secara mandiri atau melalui WA CS. Jika via WA, Admin bisa mengeklik Tambah Tiket Baru atas nama klien tersebut di dashboard Admin.
  2. Dispatch (Tugaskan): Admin membaca keluhannya, masuk ke Menu Helpdesk > Tiket Bantuan, lalu klik tombol Tugaskan Teknisi. Admin memilih Teknisi mana (Misal: "Budi") yang sedang menganggur untuk berangkat ke lokasi.
  3. Notifikasi Teknisi: Teknisi (Budi) tidak perlu masuk PC. Ia cukup membuka Link Aplikasi Mobile Teknisi di HP Android/Sub-domain yang telah disediakan.
  4. Esekusi Lapangan: Di HP Teknisi, akan muncul satu Pekerjaan Baru (Job). Ia mengklik Accept. Setelah selesai memanjat tiang atau melakukan perbaikan, Teknisi tersebut mengklik tombol Selesai.
  5. Konfirmasi Lunas: Setelah Teknisi menekan selesai, tiket tersebut di layar Admin Pusat akan otomatis berpindah status dari In Progress menjadi Selesai. Pelanggan Klien pun bisa melihat laporannya telah ditanggulangi. Sangat efisien!

8. API & Notifikasi

Menyambungkan WhatsApp / Telegram

Buka Settings -> Profil Aplikasi.

  • Fonnte (WhatsApp): Masukkan API Token dari fonnte.com untuk mengirim pengingat tagihan ke nomor WA Pelanggan (Premium Only).
  • Tripay/Midtrans: Isikan M-ID, Server Key, dan PIN untuk integrasi Gateway Pembayaran Otomatis.

9. Keamanan Akun & Backup Data

Bagaimana Cara Mengubah Password Admin & Staff?

Agar server tetap aman dari pihak tak bertanggung jawab, jadikan "Update Password" sebagai kebiasaan wajib.

  • Untuk Admin Utama: Ubah username super-admin atau password saat ini dengan mengeklik foto profil meluncur di pojok layar, masuk ke Menu Edit Profil.
  • Untuk Teknisi Lapangan / Staff CS: Admin pusat harus mendaftarkan mereka terlebih dahulu melalui Master Data -> Daftar Staff/Teknisi. Jika pegawai keluar, **segera klik ikon tong sampah / blokir** akun mereka agar layar Portal Teknisi Mobile di HP pegawai lama putus seketika, mencegah pencurian data pelanggan.
Panduan Backup & Restore Database Master

Berbeda dengan layanan Cloud Hosting biasa, MikroFun menyimpan seluruh profil Mikrotik, tagihan midtrans, dan ODP Anda di dalam satu entitas Database SQL Server ini. Data ini adalah asuransi bisnis Anda.

  • Backup Berkala Mutlak: Arahkan kursor ke Settings -> Profil Aplikasi & Database. Cari tombol Download Backup (.SQL) lalu simpan file unduhan hasil export tersebut ke komputer lokal Anda maupun ke Flashdisk. Lakukan setidaknya *tiap Seminggu Sekali* atau sebulan sekali.
  • Manfaat Backup: Jika VPS Linux/Windows Anda diserang Ransomware atau Hard disk mikrotik rusak parah, Anda dpt membeli server baru, kemudian sekadar mengupload file `.SQL` tersebut melalui opsi Restore Database. Maka dalam detik itu juga seluruh tagihan pelanggan kembali normal.
  • Tips: Jangan mencentang Auto-Backup jika Disk Space Mikrotik Anda terbatas. Unduh secara manual jauh lebih dianjurkan bagi RT-RW Net Kecil.
