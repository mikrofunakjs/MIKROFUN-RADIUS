Telegram Bot Integrasi

Konfigurasi notifikasi Telegram untuk pemberitahuan pembayaran, backup, dan peringatan sistem.

Konfigurasi Bot

Panduan Setup Bot Telegram

  • Buat Bot Baru di @BotFather

    Buka Telegram, cari @BotFather lalu chat /newbot. Ikuti langkahnya sampai mendapatkan sebuah HTTP API Token pajang. Paste Token tersebut di kolom "Bot Token API".

  • Mulai Percakapan dengan Bot Anda

    Setelah bot dibuat, cari username bot Anda di Telegram lalu klik Start atau kirim perintah /start. Ini wajib dilakukan agar bot memiliki izin untuk mengirim pesan kepada Anda.

  • Dapatkan Chat ID Anda

    Chat ID adalah alamat ID Unik target penerima pesan. Jika pesannya ingin dikirim ke Telegram Pribadi Anda, cari bot @userinfobot lalu klik Start. Bot itu akan membalas dengan deretan angka "Id". Paste Id tersebut di kolom "Chat ID".

    Tips Grup: Jika ingin notifikasi masuk ke Grup, gabungkan bot Anda ke Grup tersebut, panggil botnya, dan cek Chat ID grupnya (Biasanya diawali dengan tanda minus -).

  • Simpan dan Testing

    Klik Simpan, kemudian klik tombol Test Telegram. Jika Anda menerima notifikasi "MikroFun Test Message" di Telegram, maka persiapannya sudah selesai!

Fitur & Manfaat Telegram Bot

Notifikasi Pembayaran Real-Time

Laporan instan ke HP Bapak saat ada pelanggan membayar tagihan atau membeli voucher melalui Tripay/Midtrans. Tidak perlu refresh aplikasi terus-menerus.

Auto-Backup Cloud Database

Bot akan mengirimkan file SQL konfigurasi langsung ke obrolan Bapak. Jika VPS rusak atau error, data pelanggan tetap aman di awan Telegram Bapak.

Peringatan Dini Router Offline

Sistem memonitor jaringan secara aktif. Jika Mikrotik terputus atau mati listrik, bot akan otomatis mengirim peringatan sebelum pelanggan marah/komplain.

Laporan Pendaftaran Pelanggan Baru

Admin pusat langsung mendapat pemberitahuan saat ada penambahan ODP pelanggan PPPoE atau Hotspot yang dimasukkan ke sistem.

Laporan Harian (Daily Report)

Menerima ringkasan mutasi pendapatan harian dan statistik total pelanggan aktif otomatis ke Telegram tanpa harus membuka layar komputer.
