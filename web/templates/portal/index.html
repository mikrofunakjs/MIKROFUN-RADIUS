Portal Login Terpusat

Pengaturan Portal

Kelola halaman login voucher terpusat untuk semua router mikrotik Anda.

Apa itu Portal Terpusat?

Biasanya Anda harus mendesain dan mengupload file HTML halaman login hotspot (seperti login.html, gambar, logo) ke **setiap** router Mikrotik. Jika ada 10 router, maka Anda harus upload ke 10 router.

Dengan fitur **Portal Terpusat MikroFun**, Anda hanya perlu menggunakan **satu desain halaman login ini saja**. Semua router Mikrotik akan secara otomatis melempar (mengalihkan) layar login pelanggan ke MikroFun.

  • Desain lebih keren, modern, dan sangat responsif di HP.
  • Ganti background atau logo? Cukup diedit di MikroFun, semua router langsung ganti.
  • Lebih aman dari bypass/bobol Mikrotik karena proses login 100% dipusatkan di sini.

Metode Standar (IFrame) Metode Tanpa Upload
REKOMENDASI (100% WORKS): Gunakan metode ini agar URL HP tertahan di Mikrotik secara transparan (tanpa kelihatan pindah web). Suport di SEMUA hape jadul maupun baru.
Langkah 1: Siapkan Walled Garden
/ip firewall address-list add address={{ current_ip }} list=mikrofun_vpn_vps comment="VPN Server IP"
/ip hotspot walled-garden ip add action=accept dst-address-list=mikrofun_vpn_vps comment="Allow Portal via VPN"
Langkah 2: Download & Extract
Download File Portal (ZIP)

Ekstrak file yang didownload, lalu Timpa (Upload) isinya ke folder hotspot di MikroTik menggunakan menu Files Winbox. Pastikan script NAT Anda dihapus.

Metode Cepat (Tanpa Upload File)

WARNING: Metode ini merusak pendeteksian HTTPS, dan tidak bisa melacak MAC Address HP. Sering menyebabkan "Secure Connection Failed".

Script NAT Sakti

Copy script ini ke Terminal Winbox MikroTik Anda.

/ip firewall nat add action=dst-nat chain=dstnat dst-port=80 protocol=tcp to-addresses={{ current_ip }} to-ports={{ current_port }} comment="MikroFun: API-Proxy Portal" place-before=0
/ip firewall nat add action=masquerade chain=srcnat dst-address={{ current_ip }} dst-port={{ current_port }} protocol=tcp comment="MikroFun: API-Proxy Return Path"
/ip hotspot walled-garden ip add action=accept dst-address={{ current_ip }} comment="Allow Central Portal"
Pastikan API Service MikroTik aktif dan Port {{ current_port }} di VPS/Server MikroFun sudah dibuka.
