ACS — Panduan Setup TR-069

Konfigurasi yang harus diisi di router pelanggan agar bisa dikontrol dari MikroFun.

📡 URL ACS Server MikroFun

http://{{ server_ip }}:7547/acs/cwmp

* Gunakan HTTPS jika MikroFun sudah menggunakan SSL/TLS.

📋 Langkah Konfigurasi di Router Pelanggan

Login ke router pelanggan

Masuk ke halaman admin router (biasanya 192.168.1.1)

Cari menu TR-069 / CWMP / Remote Management

Biasanya ada di menu Administration, Management, atau Advanced.

Isi parameter berikut:
Parameter Nilai
ACS URL http://{{ server_ip }}:7547/acs/cwmp
Enable CWMP / TR-069 ✅ Enable / ON
Inform Enable ✅ Enable
Inform Interval 300 (detik)
ACS Username mikrofun (opsional)
ACS Password kosongkan atau isi bebas
Simpan dan tunggu 1-5 menit

Router akan otomatis check-in ke MikroFun ACS. Device akan muncul di menu ACS → Manajemen CPE.

Pastikan Port 7547 Terbuka di VPS

Jalankan perintah berikut di VPS untuk membuka port ACS:

sudo ufw allow 7547/tcp
