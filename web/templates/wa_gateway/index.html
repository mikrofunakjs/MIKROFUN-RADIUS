Panduan & Manfaat WhatsApp Gateway

Panduan Setup Mikrofun Node (Lokal)

  • Pastikan Service Berjalan

    Pilih Layanan Mikrofun Node. Status di atas harus menunjukkan "Scan untuk Menghubungkan".

  • Scan QR Code

    Buka aplikasi WhatsApp di HP Anda (Pilih Setting > Perangkat Taut > Tautkan Perangkat). Scan barcode yang muncul.

  • Selesai

    Setelah status berubah menjadi "Terhubung", sistem sudah siap mengirimkan pesan tagihan WA dari Dashboard Pelanggan.

Manfaat Fitur Gateway

Kirim Tagihan Otomatis

Kirim nota perpanjangan atau peringatan isolir langsung ke nomor WA pelanggan secara satu persatu melalui tabel data.

Tanpa Biaya Langganan API Fonnte

Dengan menggunakan opsi Mikrofun Node (Baileys), Anda tidak perlu membayar layanan bulanan/tahunan kepada Fonnte karena pengiriman berjalan mandiri di VPS.

Pengiriman Super Cepat

Karena engine WA terinstall secara lokal bersamaan dengan Mikrofun, tidak ada latensi dan jeda jaringan saat tombol 'Kirim WA' ditekan.
